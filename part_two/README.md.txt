Pre-requisite: Have Java and Maven configured in your preferred IDE

1. Clone the project
2. In the src/test/java/com.org.rise.capital.acceptancetests/AcceptanceTestSuite.java class, run the class as a JUnit Test
3. In the src/test/resources/serenities.properties, the browser config is there where you can change the browser variable to any browser of your choice. By default [chrome]

Edge = Microsoft Edge Driver
Firefox = Mozilla Firefox Driver
Chrome = Chrome Driver
